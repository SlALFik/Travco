# 🚷 Travco Auto-Ignore Target Userscript

A powerful and lightweight userscript for **Travcotools** that automates the tedious process of ignoring targets. Instead of manually clicking twice for every single village, this script introduces a single button to clean up your target list sequentially with smart delays to avoid bot-detection.

---

## 🚀 How to Install

### 1. Install a Userscript Manager
To run this script, you first need to install a userscript manager extension for your browser. Choose one of the following options:

* [Tampermonkey](https://www.tampermonkey.net/) *(Highly Recommended)* — available for Chrome, Firefox, Edge, Safari.
* [Violentmonkey](https://violentmonkey.github.io/) — an open-source alternative.
* [Greasemonkey](https://www.greasespot.net/) — for Firefox users.

### 2. Install the Script
Click on the link below to automatically load the script into your manager:

👉 **[Click here to install Travco Mass Ignore Script](https://github.com/SlALFik/Travco/blob/main/Autoclicker.js)**

> ℹ️ **Note:** Once clicked, your browser extension will open a new tab. Simply click the **Install** (or **Update**) button to activate it.

---

## 🛠️ Features & How It Works

* **Mass Delete Button:** Adds a prominent `💀 ВИДАЛИТИ ВСІ ЦІЛІ` (DELETE ALL TARGETS) button fixed to the top-right of your screen.
* **Smart Automation:** Automatically scrolls to each target, clicks the eye-slash icon, waits for the dynamic dropdown menu to render, and confirms the action.
* **Anti-Ban Delay:** Uses a built-in 1-second delay between actions to simulate human behavior and prevent server-side rate limiting or temporary bans.
